# Ordenação de Dígitos com Insertion Sort

Programa em C que lê uma sequência de até 10 dígitos digitados como texto,
valida a entrada e ordena os números em ordem crescente utilizando o
algoritmo Insertion Sort.

## Funcionalidades
- Leitura de números como string
- Validação de entrada (apenas dígitos)
- Conversão para inteiros
- Ordenação em ordem crescente

## Objetivo
Exercitar validação de dados, uso de vetores, funções e algoritmos de
ordenação em linguagem C.
